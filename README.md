# 🤖 AI Resume Screener

> An AI-powered web application that predicts the most suitable job role from a candidate's resume using Machine Learning.

🔗 **Live Demo:** https://resume-screener-mqz5.onrender.com

---

## 📌 Project Overview

This system takes a resume (PDF), extracts its text, converts it into numerical features using TF-IDF, and uses a trained Machine Learning model to predict the most suitable job role for the candidate.
